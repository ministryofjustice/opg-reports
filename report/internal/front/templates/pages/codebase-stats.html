Codebase stats

Active (non-archived) codebases and current, non-time based, stats for each.

Trivy usage and SBOM generation detection are based on seaching github workflows for matching keywords within active steps and are therefore not perfect.

{{ template "codebase-stats-table" .CodebaseData }}

Compliance levels are based on MoJ repository standards

Trivy usage is based on finding either `trivy ` within a github workflow / action file or the usage of the trivy action (`aquasecurity/trivy-action`).

SBOM usage is determined via looking for `trivy sbom` pattern or the action with an `output` attribute set to use `${name}.sbom.json` format.
